# BrightField

A simple, professional landing page website for BrightField - an Amazon storefront.

## Pages

- **Home** (`index.html`) - Main landing page with company overview and call-to-action
- **About** (`about.html`) - Information about the company and mission
- **Contact** (`contact.html`) - Contact information and support details

## Features

- Clean, modern responsive design
- Mobile-friendly navigation
- Professional color scheme
- No external dependencies (pure HTML/CSS)

## Setup

This is a static website. Simply open `index.html` in a web browser or deploy to any static hosting service:

- GitHub Pages
- Netlify
- Vercel
- Amazon S3
- Any web hosting provider

## Customization

- Update `contact@brightfield.com` in `contact.html` with your actual business email
- Replace the Amazon store links with your actual Amazon storefront URL
- Modify colors in `styles.css` (primary color: `#2c5530`)
